# Meq

Meq is 'multiple equals' a few macros for working with equality based functions and turning them from 2-arity to n-arity.

## Usage

    * (with-multiple-eq 1 1 1)
    * T
    * (with-multiple-eq 2 1 1)
    * nil

### Functions

* with-multiple-eq
* with-multiple-eql
* with-multiple-equal
* with-multiple-equalp
